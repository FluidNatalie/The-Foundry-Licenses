This repository is a collection of licencses that I have curated. They are mostly taken from people in The Foundry Modding community however it is important to state that The Foundry doesn't endorse me or this repository.

The current goal of the repository is to have differenct niche licenses that don't already exist. 

The MITime/SPL License are licenses that are focused on allowing licenes that are restrictive but have a time limit allowing them to be opened up once the author retires the project.

I also want a licence that says "yes, unless you're including it in a project or making edits that utilise AI/LLM generation in any capacity".
